If you have ARCHBLOX Bootstrapper already installed,
you may install version 1.6 to fix error ID 6.


Note that this version only works on Windows 7 or newer.

64 Bit 32 Bit


If you don't already have ARCHBLOX Studio installed, follow these instructions:


Step 1: Download ARCHBLOX Studio version-000000004.zip


Step 2: Go to your Local Appdata folder, then the Archblx folder,
then the Studio folder and then go into Versions.


If any folders do not exist, create them with exact spelling and caps.


Step 3: Create a folder in Versions NAMED EXACTLY "version-000000004", then extract the zip in there.
